statusbot
=========

## setup

1. copy `config.sample.php` to `config.php`
2. create a Main API key at uptimerobot.com
3. copy this key (starts with `u`) and replace `API_KEY` in the `$api_args` section of the config.
  ```
  'apiKey' => 'API_KEY',
  ```
